Clone the project

```bash
  git clone https://github.com/QFifteen/CyanLemon-admin.git
```

Go to the project directory

```bash
  cd cyanlemon-admin
```

Install dependencies

```bash
   npm install
```

Start the server

```bash
   npm run dev
```
