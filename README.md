## Local operation:old_key: 

Clone the project

```bash
  git clone https://github.com/QFifteen/CyanLemon-admin.git
```

Go to the project directory

```bash
  cd cyanlemon-admin
```

Install dependencies

```bash
   yarn install
```

Start the server

```bash
   yarn dev
```
